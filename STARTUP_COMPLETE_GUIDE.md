# 🚀 ПОЛНЫЙ ПЛАН ЗАПУСКА ЗАВТРА (Пошаговая инструкция)

---

## 📌 ШАГ 1: Открыть VS Code

**Что вы видите:**
- VS Code открыт
- Папка `D:\WEB_1` открыта слева (Explorer)

**Дальше:**
1. Внизу VS Code найди кнопку **"Terminal"** (или нажми `Ctrl + ~`)
2. Откроется чёрное окно (Terminal) внизу VS Code

---

## ⚡ ШАГ 2: Запустить Ollama в Terminal

**В Terminal окне (внизу VS Code) выполни:**

```powershell
& "$env:LOCALAPPDATA\Programs\Ollama\ollama.exe" serve
```

**Ждёшь пока появится:**
```
[GIN-debug] Loaded HTML Templates (0.289ms)
Listening on 127.0.0.1:11434
```

**✅ Ollama запущена!**

---

## 📂 ШАГ 3: Открыть второе Terminal окно

**В VS Code:**
1. Нажми иконку **"Split Terminal"** (два квадрата справа внизу Terminal окна)
2. Или нажми `Ctrl + Shift + ~` для нового терминала

Теперь у тебя **два окна Terminal**:
- **Слева:** Ollama работает (НЕ закрывай!)
- **Справа:** Пустое новое окно

---

## 🐍 ШАГ 4: Активировать venv и запустить приложение

**В новом Terminal окне справа:**

```powershell
# Перейти в папку проекта
cd D:\WEB_1

# Активировать виртуальное окружение
.\venv\Scripts\Activate.ps1
```

**Ты должна увидеть в начале строки:**
```
(venv) PS D:\WEB_1>
```

Теперь запусти приложение:

```powershell
python app.py
```

**Ждёшь пока появится:**
```
INFO:     Application startup complete.
INFO:     Uvicorn running on http://127.0.0.1:8000
```

**✅ Приложение запущено!**

---

## 🌐 ШАГ 5: Открыть браузер

**В браузере (Chrome, Firefox, Edge):**

1. Скопируй и вставь в адресную строку:
```
http://127.0.0.1:8000
```

2. Нажми **Enter**

**Ты увидишь:** Главную страницу сканера уязвимостей 🎨

---

## 🔍 ШАГ 6: Сканировать веб-сайт

**На странице есть несколько способов:**

### Способ 1️⃣: Сканировать URL
1. Нажми кнопку **"Сканировать URL"**
2. Введи URL сайта, например: `http://example.com`
3. Нажми **"Начать сканирование"**
4. **Ждёшь ~5 минут** (LLM генерирует рекомендации)
5. Увидишь результаты с рекомендациями от AI

### Способ 2️⃣: Сканировать HTML код
1. Нажми **"HTML/JS код"**
2. Вставь HTML код в окно
3. Нажми **"Начать сканирование"**

### Способ 3️⃣: Сканировать ZIP архив
1. Нажми **"ZIP архив"**
2. Выбери ZIP файл с проектом
3. Нажми **"Начать сканирование"**

### Способ 4️⃣: Сканировать папку проекта
1. Нажми **"Исходный код"**
2. Укажи путь к папке: `D:\WEB_1` или другая папка
3. Нажми **"Начать сканирование"**

---

## 📊 ШАГ 7: Посмотреть результаты

**После сканирования:**
- ✅ **Зелёный** = Low (низкая серьёзность)
- 🟡 **Жёлтый** = Medium (средняя)
- 🔴 **Красный** = High (высокая)
- 🔥 **Чёрный** = Critical (критичная)

**Для каждой уязвимости видишь:**
- Название проблемы
- Описание (от LLM AI)
- **Рекомендация (от LLM AI)** ← Вот это генерирует Ollama!
- Код где найдена проблема

## 🛑 ВАЖНО!

### ❌ Не закрывай:
- ❌ Terminal с Ollama (левое окно)
- ❌ Terminal с приложением (правое окно)
- ❌ Браузер

### ⏱️ Ожидание:
- **Первое сканирование:** ~5-10 минут (медленно)
- **Следующие:** ~3-5 минут (быстрее)
- Это нормально! Ollama думает 🧠

## 🚨 ЕСЛИ ЧТО-ТО СЛОМАЛОСЬ

### ❓ Ollama не запускается

**Решение:**
```powershell
# Закрыть предыдущий процесс Ollama
$proc = Get-NetTCPConnection -LocalPort 11434 -ErrorAction SilentlyContinue | Select-Object -ExpandProperty OwningProcess
if ($proc) { Stop-Process -Id $proc -Force }

# Запустить снова
& "$env:LOCALAPPDATA\Programs\Ollama\ollama.exe" serve
```

---

### ❓ Приложение не работает (ошибка при `python app.py`)

**Решение:**
```powershell
# Закрыть предыдущий процесс
$proc = Get-NetTCPConnection -LocalPort 8000 -ErrorAction SilentlyContinue | Select-Object -ExpandProperty OwningProcess
if ($proc) { Stop-Process -Id $proc -Force }

# Активировать venv
.\venv\Scripts\Activate.ps1

# Запустить снова
python app.py
```

---

### ❓ "Рекомендация недоступна" в результатах

**Причины:**
1. ❌ Ollama не запущена → Запусти её (Шаг 2)
2. ❌ Ollama не ответила за 5 минут → Обычно для больших сканирований

**Решение:**
- Перезагрузи страницу браузера
- Попробуй ещё раз сканировать

---

### ❓ Браузер показывает "Cannot GET /"

**Решение:**
```powershell
# Проверить запущено ли приложение (должно быть в правом Terminal)
# Если нет - запусти:
python app.py

# Ждёшь пока появится:
# INFO:     Uvicorn running on http://127.0.0.1:8000
```
