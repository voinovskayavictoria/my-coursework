# Разработка программного средства для автоматизированного выявления уязвимостей веб-приложений

Возможности:

- Проверка HTML по URL.
- Проверка вставленного HTML/JS-кода.
- Проверка zip-архива проекта.
- Поиск опасных паттернов в исходниках и конфигурации.
- Проверка security headers, CORS, CSP и cookies.
- Сохранение истории запусков и найденных уязвимостей в SQLite.
- Генерация человекочитаемых пояснений и рекомендаций через локальный LLM с fallback.
- Учетные записи, login/logout и история запусков по пользователю или сессии.

## Требования
* Python 3.11+
* pip

## Как запустить

1. Клонируйте репозиторий:
```
git clone https://github.com/voinovskayavictoria/my-coursework
cd repo
```

2. Создайте виртуальное окружение:
```
python -m venv venv
```

3. Активируйте виртуальное окружение:
```
venv\Scripts\activate
```

4. Установите зависимости:
```
pip install -r requirements.txt
```

5. Установите и подключите локальную LLM (Ollama):
Скачайте и установите Ollama с https://ollama.com/download, затем убедитесь, что команда `ollama` доступна в терминале.
```
ollama serve
ollama pull llama3.1:8b-instruct
```

6. Запустите сервер:
```
python app.py
```

7. Откройте в браузере:
```
http://127.0.0.1:8000
```

## Структура

- `app.py` — главный файл приложения FastAPI. Содержит все эндпоинты (сканирование URL, HTML, архива, исходников), обработку сессий, интеграцию с LLM, вызовы всех модулей анализа.
- `scanner.py` — модуль поиска уязвимостей в библиотеках через OSV.dev.
- `static_checks.py` — модуль статического анализа HTML/JS. 
- `source_scanner.py` — модуль анализа дерева исходников.
- `security_scanner.py` — модуль анализа HTTP-заголовков безопасности и конфигурационных файлов.
- `dependency_scanner.py` — модуль анализа зависимостей.
- `llm_support.py` — модуль интеграции с локальной LLM (Ollama). 
- `vuln_sources.py` — модуль обогащения CVE-информацией (дополнительные запросы к NVD для получения описаний, CVSS-оценок и ссылок).
- `models.py` — Описывает структуру таблиц в SQLite.
- `webapp/` — пакет с модулями FastAPI.
  - `__init__.py` — инициализация пакета.
  - `app_factory.py` — фабрика приложения FastAPI (создание экземпляра, подключение маршрутов и middleware).
  - `config.py` — настройки приложения (лимиты, таймауты, параметры сессий).
  - `auth.py` — маршруты и логика аутентификации (регистрация, логин, логаут, работа с пользователями).
  - `pages.py` — маршруты для веб-страниц (главная, история, авторизация, отчёты).
  - `session.py` — управление сессиями, работа с куками.
  - `state.py` — разделяемые данные и состояния приложения.
  - `ui.py` — вспомогательные функции для UI (рендеринг, форматирование).
  - `scans/` — подмодули для работы со сканированием (логика обработки URL, HTML, архивов).
- `templates/` — HTML-шаблоны для веб-интерфейса.
  - `auth.html` — страница входа и регистрации.
  - `history.html` — страница истории сканирований.
  - `index.html` — главная страница.
  - `report.html` — страница отчёта с найденными уязвимостями и рекомендациями.
- `requirements.txt` — список зависимостей (библиотеки Python для установки через `pip`).
- `run.ps1` — PowerShell-скрипт для быстрого запуска сервера.
- `README.md` — документация проекта.
